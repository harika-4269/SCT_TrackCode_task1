# SkillCraft Landing Page

A modern, responsive landing page for SkillCraft Technology built with HTML, CSS, and JavaScript.

## Features

- **Fixed Navigation Bar**: A responsive navbar that changes styling on scroll
- **Smooth Scroll Effects**: Dynamic navbar background and shadow effects
- **Multiple Sections**: Home, About, Services, and Contact sections
- **Mobile Responsive**: Optimized layout for different screen sizes
- **Clean Design**: Minimalist and professional styling

## How It Works

### Navbar Scroll Effect
The navbar automatically changes its appearance when the user scrolls down the page:
- Initially transparent
- Becomes darker with a shadow effect after scrolling 50px down
- Smooth transitions for visual appeal

## Usage

1. Open `task1.html` in your web browser
2. Navigate through the sections using the navigation menu
3. Scroll to see the navbar scroll effect in action

## File Descriptions

- **task1.html**: Contains the main page structure with navigation and content sections
- **style.css**: Handles all styling including navbar, sections, and scroll effects
- **script.js**: Implements the scroll event listener for navbar styling changes
